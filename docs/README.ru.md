# CryptoWalleTest

Встраиваемый платежный модуль для криптовалютных операций;
Реализован на **Go** с использованием **PostgreSQL**;

## Описание

**CryptoWalleTest** - серверный модуль вывода криптовалют;
Модуль управляет балансами пользователей;
Обрабатывает заявки на вывод средств;
Гарантирует консистентность данных при конкурентных запросах;

Начальная валюта - `USDT`;
Архитектура допускает расширение на другие криптовалюты;
Каждая валюта хранится как отдельная запись баланса;

## Возможности

- **Вывод средств** - создание и отслеживание заявок;
- **Подтверждение** - ручное подтверждение заявок перед исполнением;
- **Леджер** - журнал всех финансовых операций через `ledger_entries`;
- **Идемпотентность** - защита от дублирования через `idempotency_key`;
- **Консистентность** - транзакции PostgreSQL исключают двойное списание;
- **Авторизация** - Bearer-токен из переменных окружения;
- **Валидация** - проверка входных данных на уровне API;
- **Логирование** - структурные логи создания, ошибок и подтверждения;

## API

- `POST /v1/withdrawals` - создание заявки на вывод;
- `GET /v1/withdrawals/{id}` - получение информации о заявке;
- `POST /v1/withdrawals/{id}/confirm` - подтверждение заявки;

## Стек

- **Go** 1.22+ - язык реализации;
- **PostgreSQL** 14+ - хранение данных и транзакции;
- **Docker Compose** - локальный запуск и развертывание;
- **REST API** - интерфейс взаимодействия;

## Запуск

```sh
docker compose up --build
```
ИЛИ
```sh
make start
```

Сервер запускается на порту из переменной `APP_PORT`;
Токен авторизации задается через `APP_AUTH_TOKEN`;

## Тесты

```sh
make test
```

Покрытие включает успешные и ошибочные сценарии;
Тесты идемпотентности и конкурентного доступа;
